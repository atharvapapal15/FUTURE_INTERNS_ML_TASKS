# Task 1: AI-Powered Sales Forecasting

## 🎯 Objective
The goal of this project was to build a time-series forecasting model to predict future sales trends. This helps businesses optimize inventory, staffing, and marketing strategies.

## 🛠️ Tools & Technologies
- **Language:** Python
- **Library:** Prophet (by Facebook) for time-series forecasting
- **Environment:** Jupyter Notebook (`forecast.ipynb`)

## 📈 Methodology
1. **Data Preprocessing:** Cleaning the historical sales data and formatting it for the Prophet model.
2. **Trend Analysis:** Identifying seasonal patterns and growth trends.
3. **Forecasting:** Generating 60-day or 1-year sales predictions based on historical patterns.
4. **Visualization:** Creating dashboards to display predicted vs. actual sales.

## 📁 Included Files
- `forecast.ipynb`: The complete source code and analysis.
- `Future intern task 1 report.pdf`: Detailed documentation of findings and results.
- `Screenshots/`: Visual evidence of the forecasting model outputs.

## 🏆 Key Findings
- Successfully modeled the underlying seasonality in the sales dataset.
- Provided actionable insights for future business growth based on forecasted peaks.
